# CyberGuard Elite

Premium Ethical Cybersecurity Training Platform

## Auto Setup (run once in PowerShell)
See `setup-cyberguard.ps1`

## Manual Setup
1. Install PostgreSQL and create a database named `cyberguard`
2. Copy `.env.example` to `.env` and update `DATABASE_URL`
3. Run `npm run setup`
4. Run `npm run dev`
